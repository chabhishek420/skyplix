# Phase 9 Plan 01: Tenant Context Foundation

## Objective

- **What:** Add tenant context primitives and middleware entry points in the HTTP layer.
- **Why:** Multi-tenant isolation needs a single source of truth for tenant identity at request time.
- **Output:** Tenant-aware request context wiring plus tests for resolution and rejection paths.
