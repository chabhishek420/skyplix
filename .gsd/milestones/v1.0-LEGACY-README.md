# v1.0 Milestone (Legacy)

This directory contains the v1 planning structure for the SkyPlix TDS project. It has been superseded by GSD v2 database-backed planning.

## Migration Notes

- All Phase 1-7 planning has been consolidated into v2 milestones and slices
- The v1 roadmap, context, and requirements are preserved in `.gsd/ROADMAP.md`, `.gsd/PROJECT.md`, and `.gsd/REQUIREMENTS.md`
- v2 uses SQLite (`gsd.db`) as the source of truth; the filesystem (M001/, M002/, etc.) is generated from the database
- To view/edit planning: use `/gsd` commands in a GSD session

## Key Artifacts Preserved

- **M001-CONTEXT.md**: Milestone brief (now superseded by `.gsd/PROJECT.md`)
- **M001-ROADMAP.md**: Phase breakdown (now superseded by `.gsd/ROADMAP.md`)
