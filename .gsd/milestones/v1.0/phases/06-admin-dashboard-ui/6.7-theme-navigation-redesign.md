# Plan 6.7: Foundation & Navigation Redesign

## Objective
Migrate the Admin UI from the "Indigo Dark" theme to the professional "Original Clean White" aesthetic. This plan establishes the new design system baseline and reskins the primary navigation elements (Sidebar and TopBar).

## Context
- `admin-ui/src/index.css` — Global styles and Tailwind v4 theme
- `admin-ui/src/components/Sidebar.tsx` — Main nav component
- `admin-ui/src/components/TopBar.tsx` — Header component
- `.gsd/phases/06-admin-dashboard-ui/6-RESEARCH-REDESIGN.md` — Aesthetic findings
