## Lessons Learned
- **Promauto is safer**: Using `promauto` for metrics avoids common registration panics in high-concurrency Go apps.
- **Docker CA Certs**: Reverse proxies inside containers require `ca-certificates` (Alpine) to handle HTTPS targets.
- **k6 Scenario Modeling**: Ramping arrival rate is superior to constant VUs for testing burst traffic in TDS systems.
